🧬 비유 — 의학적 관점

국면 = 환자의 활력징후 점수 (1~5 등급, 단순 위험도)
캐스케이드 = 환자의 병명/진단 (감기/폐렴/심근경색)

같은 활력징후 4등급이라도:
• 감기 (≈ narrative_scare) → 휴식만으로 OK
• 폐렴 (≈ real_stress_brewing) → 항생제 필요
• 심근경색 초기 (≈ mechanical_cascade) → 즉시 응급실

→ 활력징후만 보고 처방하면 위험. 둘 다 봐야 정확한 처치.
